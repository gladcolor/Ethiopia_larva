## Data

- **Source imagery:** Jigjiga 60 m split tiles, acquisition 2026-05-03
- **Patches:** 1518 verified sites (`Jigjiga_20260503_slipt_60m_detection_v2_verified.shp`)
- **Predictions:** generated by the LightGBM model in `density_prediction/larval_density_prediction20260503.ipynb`
- **Range:** predicted larvae count from 0 to ~304 per patch (38 zero-prediction patches included)
